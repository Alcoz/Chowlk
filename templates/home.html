OWL Code Generation Service

Chowlk is a web service that automatically generates the OWL code from your Ontological Diagrams.

In order to use our service you should use Drawio as the diagraming tool, and follow our specification to conceptualize your ontologies. Once it is finished, export it as XML, and load it in our drag and drop tool.

View Specification

Previous steps before using the service


  1. The ontological diagram always should include the namespace block, indicating all the prefixes and namespaces used in the ontology.
  2. The metadata block must contain the actual annotation property elements, aliases are not allowed. For instance, the user should type owl:versionInfo instead of "version" to indicate the current version of the ontology.
  3. The user must ensure that the source and target of all the arrows in the model are connected to a block. This means that in order to represent an owl:subPropertyOf relationship the user should use the form 1 for this construct.
